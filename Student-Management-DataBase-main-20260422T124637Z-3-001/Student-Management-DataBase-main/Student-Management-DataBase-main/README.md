**JDBC Student Management System**
A simple console-based Java application that demonstrates how to use JDBC (Java Database Connectivity) to interact with a relational database.
This mini project performs basic CRUD operations and demonstrates concepts like PreparedStatement, ResultSet, Batch Processing, and Stored Procedures.

**Project Features**
The application allows users to:
Add a student record
View all student records
Update student marks
Delete a student record
Insert multiple students using Batch Processing
Use a Stored Procedure to insert records
Safely manage database resources using try-with-resources

**Technologies Used**
Java
JDBC
PostgreSQL
Console-based interface
