**Advanced Quiz Application:**
A console-based Java quiz application that demonstrates object-oriented programming concepts such as inheritance, polymorphism, method overriding, and abstract classes while building an interactive multiple-choice quiz system.
In this version of the quiz application, different types of questions are introduced with different scoring rules. Basic questions award normal points, while advanced questions include negative marking, making the quiz more realistic and challenging.

**Project Features**
This project helps demonstrate how polymorphism allows multiple types of questions to be stored and processed using a single reference type, improving the flexibility and extensibility of the program.
